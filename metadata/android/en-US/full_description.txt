Open doors using HTTPS/SSH/MQTT or Bluetooth.

Open or close doors via several supported queries. Also show if a door is open or closed.
Send HTTPS queries, execute commands via SSH or publish/subscribe MQTT messages.

Features:

* Manage multiple doors
* QR-Code support
* Door setup backup
* Custom door state pictures
* Pin TLS certificates and set SSH keys
* Auto-select door setup by WiFi SSID
* Display returned text from door
