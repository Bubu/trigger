Open doors using HTTPS/SSH/MQTT or Bluetooth.

Open or close doors, ring the door bell and show the door state.
This is done via HTTPS queries, SSH commands or via MQTT messages.

Features:

* Manage multiple doors
* QR-Code support
* Door setup backup
* Custom door state pictures
* Pin TLS certificates and set SSH keys
* Auto-select door setup by WiFi SSID
* Display returned text from door
